## 웹 기초 지식
 **인터넷** 
 - 글로벌 컴퓨터 네트워크 시스템

 **프로토콜**
 - 컴퓨터 내부 또는 컴퓨터 사이에서 데이터의 교환 방식을 정의하는 규칙 체계
 - ex. HTTP(HyperText Transfer Protocol)

  
 **웹**
 - wolrd wide web
 - HTML 언어로 작성 -> URL로 위치 지정 -> HTTP로 데이터 전송
 * URL의 구조
    - 프로토콜://도메인이름:포트/경로/파라미터
